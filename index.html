Welcome the Adoption Site

The application will track the names of the birth mother, adoptive mother, and adoptee's information


You can list the following information:

adoptee: first name, last name, gender, region, birth year, adoption year, and age at adoption.
adoptive mother: first name, last name, and years waiting. birth mother: first name, and last name.

You can find the code for this project via github https://djimd.github.io/Adoption
